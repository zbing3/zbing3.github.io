python计算文件md5

Python中，用于加密的md5方法在hashlib模块中，文件的MD5校验码是根据文件的内容生成的信息摘要，方法如下:

from hashlib import md5

def md5_file(name):
    m = md5()
    a_file = open(name, 'rb')    #使用二进制格式读取文件内容
    m.update(a_file.read())
    a_file.close()
    return m.hexdigest()

if __main__ == '__init__':
