sublime 支持开发 coffeescript compass sass

没啥难的看我的节奏：

使用Shift + command + P调出命令面板：

输入 install 调出 Package Control: Install Package 选项，按下回车

在列表中分别找到 CoffeeScript，compass，sass和sass build按下回车进行安装

重启Sublime Text 2使之生效
